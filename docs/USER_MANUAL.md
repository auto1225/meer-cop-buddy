## 1. MeerCOP이란?

MeerCOP은 노트북/컴퓨터의 도난을 방지하고, 도난 시 기기를 추적·복구할 수 있도록 돕는 보안 감시 앱입니다.

**주요 기능:**
- 🎥 카메라 모션 감지 — 움직임이 감지되면 자동 촬영 및 경보
- ⌨️ 키보드/마우스 입력 감지 — 무단 사용 시 즉시 알림
- 🔌 전원/USB 변화 감지 — 전원 케이블이나 USB 기기 변경 시 경보
- 📍 실시간 위치 추적 — GPS를 통한 기기 위치 확인
- 📷 원격 카메라 스트리밍 — 스마트폰에서 실시간 카메라 영상 확인
- 🔇 위장 모드 — 화면을 검게 하여 감시 중임을 숨김
- 📱 스마트폰 연동 — 어디서든 기기 상태 확인 및 원격 제어

## 2. 시작하기

### 2.1 시리얼 넘버 등록

1. 앱을 실행하면 **시리얼 넘버 입력 화면**이 나타납니다.
2. 제공받은 **12자리 시리얼 넘버**를 4자리씩 3개 입력란에 입력합니다.
   - 예: `HKXQ` - `XG7W` - `54NY`
3. 시리얼 넘버는 대소문자를 구분하지 않습니다.

### 2.2 기기 이름 설정

1. 시리얼 넘버 아래의 **기기 별칭** 입력란에 원하는 이름을 입력합니다.
   - 예: "내 노트북", "사무실 PC"
2. 여러 기기를 등록할 경우 구분이 쉬운 이름을 사용하세요.

### 2.3 기억하기 기능

- **"기억하기(Remember Me)"** 체크박스를 선택하면, 다음 실행 시 시리얼 넘버와 기기 이름이 자동으로 입력됩니다.
- 공용 컴퓨터에서는 체크를 해제하는 것을 권장합니다.

> ⚠️ **로그아웃** 시 저장된 인증 정보가 완전히 삭제됩니다.

## 3. 컴퓨터(노트북) 앱 사용법

### 3.1 메인 화면 구성

| 영역 | 설명 |
|------|------|
| **상단 바** | 메뉴(☰), 앱 이름, 알림(🔔), 기기 추가(+) |
| **기기 선택** | 현재 선택된 기기 이름 + 설정(⚙️) 버튼 |
| **마스코트** | 감시 상태에 따라 변하는 MeerCOP 캐릭터 |
| **상태 아이콘** | 카메라(📷), 네트워크(📶), 사운드(🔊) 상태 |
| **상태 메시지** | 현재 감시 상태를 텍스트로 표시 |
| **감시 토글** | 감시 시작/중지 버튼 |

### 3.2 감시 시작/중지

1. 메인 화면 하단의 **감시 토글 버튼**을 클릭합니다.
2. **감시 시작** 시:
   - 마스코트가 감시 중 상태로 변경됩니다.
   - 활성화된 센서들이 동작을 시작합니다.
   - 카메라가 연결되어 있으면 자동으로 카메라 감시도 시작됩니다.
3. **감시 중지** 시:
   - PIN 입력이 필요할 수 있습니다(설정에 따라 다름).
   - 모든 센서 감시가 중지됩니다.

### 3.3 센서 설정

메뉴(☰) 또는 설정(⚙️)에서 센서를 개별적으로 ON/OFF할 수 있습니다.

| 센서 | 설명 | 기본값 |
|------|------|--------|
| **카메라** | 카메라 모션 감지 | ON |
| **키보드** | 키 입력 감지 | ON |
| **마우스** | 마우스 움직임 감지 | ON |
| **USB** | USB 기기 연결/해제 감지 | ON |
| **전원** | 충전 케이블 연결/해제 감지 | ON |
| **덮개(리드)** | 노트북 덮개 열림/닫힘 감지 | OFF |
| **마이크** | 소리 감지 | OFF |

### 3.4 경보 설정

#### PIN 설정
- **기본 PIN**: `1234`
- 경보 해제 시 입력해야 하는 4자리 숫자입니다.
- 설정에서 변경할 수 있습니다.

#### PC에서 PIN 필요 여부
- **활성화(기본)**: 컴퓨터에서 경보 해제 시 PIN 입력 필요
- **비활성화**: PIN 없이 즉시 해제 가능

#### 경보음 선택
다양한 경보음 중 선택할 수 있습니다:

| 경보음 | 설명 |
|--------|------|
| 🚨 경찰 사이렌 | 경찰차 사이렌 소리 |
| 🔔 보안 경보 | 일반 보안 알람 소리 |
| ⚠️ 공습 사이렌 | 긴급 상황 사이렌 |
| 🚷 침입자 경보 | 침입 감지 알림음 |
| 🆘 비상 경보 | 비상 상황 알림음 |
| 🚗 차량 경보 | 자동차 경보음 |
| 📢 긴급 경적 | 큰 경적 소리 |
| 🛡️ 도난 방지음 | 도난 방지 전용 음 |
| 📣 대형 클랙슨 | 대형 클랙슨 소리 |
| ⚡ 트리플 경보 | 3중 복합 경보음 |

#### 카메라 민감도
| 설정 | 설명 |
|------|------|
| **민감** | 작은 움직임도 감지 (변화율 약 3%) |
| **보통(기본)** | 일반적인 움직임 감지 (변화율 약 15%) |
| **둔감** | 큰 움직임만 감지 (변화율 약 80%) |

#### 마우스 민감도
| 설정 | 설명 |
|------|------|
| **민감** | 5px 이상 움직임 감지 |
| **보통(기본)** | 30px 이상 움직임 감지 |
| **둔감** | 100px 이상 움직임 감지 |

### 3.5 위장 모드

위장 모드를 활성화하면 화면이 완전히 검게 변합니다.

- **목적**: 감시 중임을 제3자에게 숨기기 위함
- **활성화**: 설정에서 "위장 모드" 토글 ON
- **해제**: PIN 입력 또는 스마트폰에서 원격 해제
- 위장 모드에서도 모든 센서는 정상 동작합니다.

### 3.6 카메라 스트리밍

웹캠이 연결된 상태에서 스마트폰으로 실시간 영상을 전송할 수 있습니다.

- 카메라가 연결되면 자동으로 스트리밍 준비 상태가 됩니다.
- 스마트폰 앱에서 카메라 보기를 시작하면 실시간 영상이 전송됩니다.
- 카메라 연결이 해제되면 스트리밍이 자동 중지됩니다.
- 카메라가 재연결되면 스트리밍이 자동으로 재개됩니다.

## 4. 스마트폰 앱 사용법

### 4.1 메인 화면 구성

| 영역 | 설명 |
|------|------|
| **상단 바** | 메뉴, 앱 이름, 알림, 기기 추가 |
| **기기 선택** | 감시 중인 기기 선택 드롭다운 |
| **기기 상태** | 온라인/오프라인, 카메라, 네트워크 상태 |
| **빠른 액션** | 카메라 보기, 위치 확인, 네트워크 정보 등 |
| **경보 상태** | 현재 경보 발생 여부 표시 |

### 4.2 실시간 카메라 보기

1. 메인 화면에서 **카메라 아이콘**을 탭합니다.
2. 컴퓨터의 웹캠 영상이 실시간으로 표시됩니다.
3. 영상은 WebRTC를 통해 P2P로 전송되므로 지연이 최소화됩니다.
4. 카메라 보기를 종료하려면 닫기 버튼을 탭합니다.

> 💡 **팁**: 컴퓨터의 카메라가 연결되어 있어야 실시간 보기가 가능합니다. 카메라 상태는 상태 아이콘에서 확인할 수 있습니다.

### 4.3 원격 경보 해제

경보가 발생하면 스마트폰에 알림이 표시됩니다.

1. 경보 알림을 확인합니다.
2. **"경보 해제"** 버튼을 탭합니다.
3. 컴퓨터의 경보음이 즉시 중지됩니다.

> ⚠️ 스마트폰에서의 원격 해제는 PIN 입력 없이 즉시 해제됩니다.

### 4.4 원격 설정 변경

스마트폰에서 컴퓨터의 설정을 원격으로 변경할 수 있습니다:

- **PIN 변경**: 경보 해제용 PIN 숫자 변경
- **경보음 변경**: 다른 경보음 선택
- **센서 ON/OFF**: 개별 센서 활성화/비활성화
- **카메라 민감도**: 모션 감지 민감도 조절
- **마우스 민감도**: 마우스 감지 민감도 조절
- **위장 모드**: 원격으로 위장 모드 활성화/해제

설정 변경은 실시간으로 컴퓨터에 반영됩니다.

### 4.5 위치 확인

1. **위치 확인** 버튼을 탭합니다.
2. 컴퓨터에 위치 정보 요청이 전송됩니다.
3. 잠시 후 지도에 컴퓨터의 현재 위치가 표시됩니다.

> ⚠️ 컴퓨터가 온라인 상태이고, 위치 서비스가 활성화되어 있어야 합니다.

### 4.6 네트워크 정보 확인

1. **네트워크 정보** 버튼을 탭합니다.
2. 컴퓨터의 네트워크 상태를 확인할 수 있습니다:
   - 연결 유형 (WiFi, 유선 등)
   - 다운로드 속도
   - 응답 시간(RTT)
   - 네트워크 품질

## 5. 경보 및 알림

### 5.1 경보 종류

| 경보 유형 | 트리거 조건 | 아이콘 |
|-----------|-------------|--------|
| **카메라 모션** | 카메라 앞 움직임 감지 | 📷 |
| **키보드** | 키보드 키 입력 감지 | ⌨️ |
| **마우스** | 마우스 이동 감지 | 🖱️ |
| **덮개** | 노트북 덮개 열림/닫힘 | 💻 |
| **전원** | 전원 케이블 연결/해제 | 🔌 |
| **충격** | 기기에 물리적 충격 | 💥 |
| **이동** | 기기 위치 변경 | 📍 |

### 5.2 경보 해제 방법

경보가 발생하면 다음 방법으로 해제할 수 있습니다:

#### 컴퓨터에서 해제
1. 경보 화면에서 **PIN 입력** (설정에서 PIN 필요가 활성화된 경우)
2. 올바른 4자리 PIN을 입력하면 경보가 해제됩니다.
3. PIN 필요가 비활성화된 경우, 해제 버튼 클릭만으로 해제됩니다.

#### 스마트폰에서 원격 해제
1. 스마트폰 앱의 경보 알림에서 **"경보 해제"** 버튼 탭
2. PIN 없이 즉시 해제됩니다.

#### 자동 해제
- 일부 경보는 설정된 시간이 지나면 자동으로 해제됩니다.

### 5.3 경보 사진 확인

카메라 모션 감지 경보 발생 시:
1. 감지 순간의 사진이 자동으로 촬영됩니다.
2. 촬영된 사진은 스마트폰 앱으로 실시간 전송됩니다.
3. 경보 상세 화면에서 촬영된 사진을 확인할 수 있습니다.

## 6. 센서 상세 설명

### 카메라 모션 감지
- 웹캠으로 촬영되는 영상의 프레임 간 변화를 분석합니다.
- 설정된 민감도에 따라 변화율이 임계값을 초과하면 경보가 발생합니다.
- 경보 발생 시 자동으로 사진을 촬영하여 증거로 저장합니다.

### 키보드 감지
- 키보드의 모든 키 입력을 감지합니다.
- 감시 시작 후 첫 키 입력 시 즉시 경보가 발생합니다.

### 마우스 감지
- 마우스 커서의 이동을 감지합니다.
- 설정된 민감도(px 단위) 이상의 움직임이 감지되면 경보가 발생합니다.
- 유선/무선 마우스 타입을 설정할 수 있습니다.

### USB 감지
- USB 포트에 새 기기가 연결되거나 기존 기기가 해제될 때 감지합니다.

### 전원 감지
- 충전 케이블의 연결/해제 상태 변화를 감지합니다.
- 노트북 도난 시 전원 케이블이 분리되는 상황을 감지하는 데 유용합니다.

### 덮개(리드) 감지
- 노트북 덮개의 열림/닫힘 상태 변화를 감지합니다.
- 기본적으로 비활성화되어 있습니다.

### 마이크 감지
- 주변 소리를 감지합니다.
- 기본적으로 비활성화되어 있습니다.

## 7. 도난 복구 모드

기기가 도난되었다고 판단될 경우, 도난 복구 모드가 활성화됩니다.

### 자동 동작
1. **위치 추적**: GPS를 통해 기기의 현재 위치를 주기적으로 업데이트합니다.
2. **사진 전송**: 카메라로 주기적으로 사진을 촬영하여 스마트폰으로 전송합니다.
3. **경보 유지**: 경보음이 계속 재생됩니다.

### 스마트폰에서 확인
- 스마트폰 앱에서 실시간으로 기기 위치를 지도에서 확인할 수 있습니다.
- 촬영된 사진을 통해 도난범을 식별할 수 있습니다.

## 8. 다중 기기 관리

하나의 시리얼 넘버 계정으로 여러 기기를 등록하고 관리할 수 있습니다.

### 기기 추가
1. 새 컴퓨터에서 MeerCOP 앱을 실행합니다.
2. **동일한 시리얼 넘버**로 로그인합니다.
3. **다른 기기 이름**을 설정합니다.

### 기기 전환
- 스마트폰 앱 상단의 기기 선택 드롭다운에서 원하는 기기를 선택합니다.
- 선택된 기기의 상태와 설정이 표시됩니다.

### 동시 감시
- 등록된 모든 기기가 동시에 감시될 수 있습니다.
- 경보는 발생한 기기의 정보와 함께 스마트폰에 전달됩니다.

## 9. 문제 해결(FAQ)

### Q: 시리얼 넘버를 잊어버렸어요.
**A:** 관리자에게 문의하여 시리얼 넘버를 재발급받으세요.

### Q: 카메라가 감지되지 않아요.
**A:** 다음을 확인하세요:
1. 웹캠이 물리적으로 연결되어 있는지 확인
2. 다른 앱에서 카메라를 사용 중인지 확인
3. 브라우저에서 카메라 권한을 허용했는지 확인
4. 카메라 드라이버가 정상 설치되었는지 확인

### Q: 경보음이 들리지 않아요.
**A:** 다음을 확인하세요:
1. 컴퓨터 볼륨이 켜져 있는지 확인
2. 브라우저에서 소리 재생이 차단되지 않았는지 확인
3. 경보음 설정에서 올바른 경보음이 선택되었는지 확인

### Q: 스마트폰에서 카메라 영상이 보이지 않아요.
**A:** 다음을 확인하세요:
1. 컴퓨터의 카메라 상태 아이콘이 활성화(초록색)인지 확인
2. 컴퓨터가 온라인 상태인지 확인
3. 스마트폰과 컴퓨터가 모두 인터넷에 연결되어 있는지 확인
4. 카메라 보기를 닫고 다시 열어보세요

### Q: 위치가 정확하지 않아요.
**A:** GPS 기반 위치는 환경에 따라 오차가 발생할 수 있습니다:
- 실내에서는 정확도가 떨어질 수 있습니다
- WiFi 기반 위치 추정이 사용될 수 있습니다
- 위치 서비스 권한이 "정확한 위치"로 설정되어 있는지 확인하세요

### Q: 감시 중에 컴퓨터가 느려져요.
**A:** 다음을 시도하세요:
1. 카메라 민감도를 "둔감"으로 변경
2. 불필요한 센서를 비활성화
3. 다른 무거운 프로그램을 종료

### Q: "기기 오프라인"으로 표시돼요.
**A:** 컴퓨터에서 MeerCOP 앱이 실행 중이고 인터넷에 연결되어 있는지 확인하세요. 앱이 실행 중이라면 잠시 후 자동으로 온라인 상태로 변경됩니다.
